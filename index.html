Plano IoT

Monitoreo con sensores

Capturamos variables como temperatura, humedad y calidad del aire con sensores, conectados al ESP32.

Datos accesibles

La información recolectada es enviada, procesada y presentada en tiempo real en una interfaz web clara y funcional.

Infraestructura escalable

El sistema combina hardware modular y conectividad Wi-Fi, permitiendo extender el monitoreo en diferentes espacios físicos.

Dispositivos y módulos utilizados Elementos electrónicos que componen e integran el sistema IoT

Tarjeta de Expansión

Shield para 30 Pines para Modulo ESP32.

ESP32 DevKit V1

Microcontrolador WiFi potente, versátil, compacto, ideal para IoT.

Pantalla lcd 16x2

Muestra información usando 16 caracteres por fila, controlada por pines.

Módulo Relé 5v

Interruptor electrónico, controla dispositivos, seguro, ideal para automatización.

Cable - Protoboard

Cable jumper, conexión rápida protoboard Arduino.

Transistor 2n2222

Transistor NPN controla relé, amplifica señal desde microcontrolador.

ConclusionesResumen del proyecto y aprendizajes clave

Integración efectiva

Hardware & Software

La conexión entre sensores, microcontrolador y la interfaz web fue lograda con éxito, demostrando la funcionalidad del sistema IoT.

Transmisión en red

Conectividad

Se logró enviar y visualizar datos ambientales a través de Wi-Fi, garantizando monitoreo sin cables y en tiempo cuasi-real.

Interfaz accesible

Diseño Web

Se desarrolló una interfaz clara y adaptable para visualizar datos relevantes, facilitando la supervisión por parte del usuario final.

Escalabilidad

Futuras mejoras

El sistema puede expandirse con nuevos sensores y funcionalidades, adaptándose a distintos entornos y requerimientos.
